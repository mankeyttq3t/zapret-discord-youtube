## üéÆ Usage

Once the bot is running, you can start using it in your Discord server. Here are some basic commands:

- **Play a Video**: Use the command `!play <YouTube URL>` to play a video.
- **Pause**: Use `!pause` to pause the current video.
- **Resume**: Use `!resume` to continue playback.
- **Stop**: Use `!stop` to stop the video and clear the queue.
- **Queue**: Use `!queue` to see the list of upcoming videos.

### Example Commands

```plaintext
!play https://www.youtube.com/watch?v=dQw4w9WgXcQ
!pause
!resume
!stop
!queue
```
